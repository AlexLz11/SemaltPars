ul.catalog-section-list-tile-list - каталог с подкаталогами на гл странице
    a.catalog-section-list-item-link - ссылки на подкаталоги на гл странице

div.bx_catalog_tile - контейнер с подкаталогами на подстраницах (в том числе на стр с товарами)
    h1.bx_catalog_tile_category_title - контейнер с именем категории на подстраницах
        a - имя категории внутри тега
    h2.bx_catalog_tile_title - элементы с именами подкаталогов и ссылками на них (отсутствует на страницах с товаром)
        a - ссылки на подкаталоги

div.wrapper_product_items   - содержит товары, есть только на страницах с товарами
    div.product-item     - элементы с товарами на странице
        div.product-item-title   - элемент с названием товарами
            a.text - название товара
        div.product-item-price-current   - содержит цену с точкой (41.40 ₽), стрип по ' \n₽'

div.bx-pagination    - наличие пагинации на странице, содержит внутри количество <li> - 2 = макс страниц пагинации
    li - кол во элементов. ссылка на 2 страницу = link + ?PAGEN_1=2
